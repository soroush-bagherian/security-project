# security project

#### features :
  - authentication with linux system users
  - dashbord : View files that the authenticated user has access to
  - store login attamp
  - monitoring for most used login info
  - auto fill iptables rule script for block ips with given range ip file

### requirement

* php 8.x.x
* redis

### Installation
- install vendor
- install nodemodules
